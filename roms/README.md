# ROMs Directory

This directory is the designated location for Sega Genesis/MegaDrive ROM files used in this project.

## Files to Place Here

Place your legally obtained ROM backups in this folder using the following naming convention:

- `Sonic 3.md` — Sonic The Hedgehog 3 (Sega Genesis/MegaDrive)
- `Sonic & Knuckles.md` — Sonic & Knuckles (Sega Genesis/MegaDrive)
- `Sonic 3 & Knuckles.md` — Sonic 3 & Knuckles (Combined Lock-On ROM)

## Important Legal Notice

Copyrighted ROM files are **not** included in this repository. You must provide your own legally obtained ROM backups dumped from cartridges you own.

See [GUIDE_ADD_ROMS.md](../GUIDE_ADD_ROMS.md) for full instructions on how to add and use these ROM files with emulators, flash carts, and ROM hacking tools.
